02 — The three bars

Every candidate fails at least one.

Bar one
Mathematical & probabilistic reasoning
Expected value, conditional probability, combinatorics, game theory, mental arithmetic under pressure. The physiotherapy curriculum does not cover this. Build it deliberately, in parallel with everything else, starting now.
Bar two
Algorithms & competitive programming
Not LeetCode medium — Codeforces Div 1 level. Dynamic programming, graph algorithms, segment trees, string algorithms, game theory. This is an 18–24 month build. The only way through is through.
Bar three
Systems engineering depth
Memory hierarchy, lock-free data structures, CPU cache behavior, memory ordering. Ghost VM is the foundation. The order book and lock-free ring buffer extend it into HFT territory.
