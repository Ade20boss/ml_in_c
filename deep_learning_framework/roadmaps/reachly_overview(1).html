The Problem

The job market is broken for entry-level candidates

AI made it trivially easy to apply to hundreds of jobs. Recruiters know this. Your resume hits an ATS wall, gets filtered in 6 seconds, and nobody ever sees it. The game changed. The tools haven't.

01
🤖

ATS filters kill applications

75% of resumes are rejected before a human reads them. Applicant Tracking Systems filter by keyword matching — not by actual competence or fit.

02
🌊

Recruiters are overwhelmed

A single job post now receives 500–1,000+ applications. You're not being ignored — you're drowning in a flood of AI-generated CVs that all look identical.

03
🤝

Relationships open closed doors

Up to 80% of roles are filled through networks before they're publicly posted. If you don't know someone, you're always too late — and nobody teaches you how to build that network fast.

04
🧩

No tool connects all the pieces

LinkedIn, Indeed, ChatGPT for your CV, Gmail for outreach, and a spreadsheet to track it all. Nothing works together. You're managing a job search like a project with no project management.

Chrome Extension

It does the reaching out. You do the interviewing.

No copy-paste. No manual clicking. The Reachly extension runs inside your browser, acts as you on LinkedIn and Twitter/X, and executes every outreach action automatically — within safe daily limits so your account stays clean.

🧩
User installs the extension
One-click install from the Chrome Web Store. It connects to your Reachly account and reads your action queue from the dashboard. No LinkedIn API. No credentials shared. It runs entirely in your own browser session.
🔍
Reachly queues actions automatically
When Reachly finds a recruiter or relevant contact for you, it generates a personalized message and queues a connection request. You see everything before it goes out and can edit or skip any action.
⚡
Extension executes in the background
The extension navigates to each recruiter's LinkedIn profile, clicks Connect, sends the personalized note, and closes — all with human-like delays (2–8 seconds between actions) so LinkedIn's detection systems see normal activity.
🛡️
Hard safety limits protect your account
Maximum 20 connection requests per day. Actions only run during your normal active hours. Auto-pauses if LinkedIn shows any warning or captcha. Your LinkedIn account is the asset — Reachly protects it.
📊
Everything syncs to your CRM
Every action the extension takes — sent, accepted, replied, ignored — syncs back to your Reachly dashboard in real time. Follow-up reminders are scheduled automatically based on response status.

★ Killer Feature 1

Warm Path Finder

Cold outreach gets a 2–5% reply rate. A warm introduction gets 40–60%. Reachly maps your existing network against your target companies and finds who you actually know — or know through one degree — that can get you in the door.

STEP 1 — EXPORT
Export your LinkedIn connections
LinkedIn lets you download your full connections list as a CSV under Settings → Data Privacy → Get a copy of your data. Upload it to Reachly once. No API needed. Your data stays yours.
STEP 2 — MATCH
AI maps your network to target companies
Reachly cross-references your connections against your target companies. It finds people who work there now, left recently (and still have relationships), or are connected to key decision-makers.
STEP 3 — REACH
Extension sends the warm intro request
Reachly generates a warm intro request to your mutual contact, the extension sends it via LinkedIn message automatically. One conversation turns a cold application into a warm referral.

async function reachlyPipeline(user, target) {
// 1. discover via Brave Search (cached)
const recruiters = await cachedSearch(target);
// 2. map warm paths from connections CSV
const warmPaths = await findWarmPaths(user.network, target);
// 3. generate messages via Groq
const messages = await groq.generate({ recruiters, warmPaths, user });
// 4. queue for extension execution
await supabase.from('action_queue').insert(messages);
// 5. extension picks up queue, acts on LinkedIn
// chrome.runtime → content script → LinkedIn DOM
// 6. tailor resume via Gemini Flash
const resume = await gemini.tailor(user.resume, target.jd);
// 7. on interview scheduled → auto-prep
if (crm.status === 'interview') buildPrepPack(target);
}
